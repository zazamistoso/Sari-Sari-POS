# Sari-Sari Store POS 🏪

A point-of-sale (POS) system built for a small sari-sari store in the Philippines.
Built with Electron, React, and Vite.

## Features (in progress)
- 🛒 Retail and wholesale transaction modes
- 📦 Product and barcode management
- 💾 Offline-first with local database
- ☁️ Cloud sync when internet is available
- 📊 Daily and monthly sales summary

## Tech Stack
- Electron
- React
- Vite
- SQLite (local database)
- Supabase (cloud sync)
